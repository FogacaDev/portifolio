# Portfólio Pessoal

Este é o repositório do meu portfólio pessoal, desenvolvido para exibir meus projetos e habilidades em desenvolvimento web. O portfólio inclui temas dinâmicos (claro e escuro), botões interativos, e uma estrutura organizada para projetos futuros.

## Tecnologias Utilizadas

- **HTML5**: Para estruturação do conteúdo.
- **CSS3**: Para estilização e layout responsivo.
- **JavaScript**: Para interação dinâmica com o usuário.
- **Bootstrap**: Framework CSS para design responsivo e componentes interativos.
- **Git**: Para controle de versão.
- **GitHub**: Para hospedagem do código e colaboração.

## Funcionalidades

- **Tema Claro/Escuro**: Alterna entre temas claro e escuro para melhor experiência do usuário.
- **Botões de Navegação**: Interação com elementos do portfólio através de botões estilizados.
- **Configurações Personalizadas**: Adaptação do design conforme preferências do usuário.
- **Estrutura Responsiva**: Adaptável para diferentes dispositivos, incluindo desktops, tablets e celulares.
- **Componentes Bootstrap**: Utilização de componentes do Bootstrap, como botões, formulários e cards, para melhorar a experiência de navegação.
